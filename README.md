# Colors for Kids

## Overview

This task project was made as semsetral project for my school class. It's created to based on set rules that were given to me in the class. Colors for Kids is an educational cli game designed to help children learn about colors in a fun and engaging way. This interactive game provides a platform where kids can explore different colors through various activities, exercises, and games that stimulate their imagination and creativity.

## Playing the game

The goal of the game is to make the whole grid in only in blur color. By switching colors in row or column. The game is very simple and was made in few minutes of work.
