# Colors for Kids

## Overview

Colors for Kids is an educational game designed to help children learn about colors in a fun and engaging way. This interactive game provides a platform where kids can explore different colors through various activities, exercises, and games that stimulate their imagination and creativity.

## Features
- **Interactive Learning**: Children can learn colors through interactive activities that keep them engaged and entertained.
- **User-Friendly Interface**: The design is intuitive, allowing kids to navigate the game easily.
- **Variety of Games**: Includes multiple games focused on color recognition, matching, and sorting.
- **Rewards System**: Children earn rewards as they progress, encouraging them to continue learning.

## How to Play
1. **Choose a Game**: Select from various color-related games.
2. **Follow Instructions**: Each game has simple instructions that guide the child through the activity.
3. **Learn and Have Fun**: Engage with the colors and enjoy the rewards as they progress!

## Installation
To install the game, follow these steps:
1. Clone the repository: `git clone https://github.com/matbrtna/ColorsGame_PCC.git`
2. Navigate to the project directory: `cd ColorsGame_PCC`
3. Install dependencies: `npm install`
4. Start the game: `npm start`
